## {{ title }}

* Programmeertaal: Naar keuze

***Opdracht:***  
Bij deze opdracht hoort een XML-bestand. Maak een programma dat het
aangeleverde XML bestand converteert naar een JSON-bestand. Het
JSON-bestand moet het volgende formaat hebben:

```javascript
{
  "Klanten": [{
    "Bedrijf": "",
    "Voornaam": "",
    "Achternaam": "",
    "Email": "",
    "Straatnaam": "",
    "Stad": "",
    "Postcode": "",
    "Land": ""
  }]
  }      
```
